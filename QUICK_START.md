# Quick Start - 빠른 시작 가이드

## 🎵 동영상 녹화 빠른 시작

### 1️⃣ 4바퀴 완성하기
```
1. 화살표(← →)로 Circle 3까지 이동
2. 키보드로 패턴 만들기
3. 4바퀴가 돌 때까지 연주
```

### 2️⃣ 다운로드 버튼 클릭
```
우측 하단에 나타나는 📥 Download Video 버튼 클릭
```

### 3️⃣ 설정하기
```
✏️ LP Title: 타이틀 입력 (선택사항)
🎨 Background Color: 배경색 선택
📐 Video Format: Square(정사각) 또는 Mobile(세로)
```

### 4️⃣ 녹화 시작
```
Start Recording 버튼 클릭
→ 한 바퀴 자동 녹화
→ 자동 다운로드
```

## 💡 팁
- **인스타 피드**: Square + 밝은 배경
- **스토리/릴스**: Mobile + 어두운 배경
- **타이틀**: 짧고 임팩트있게!

## 📱 추천 설정

### 인스타그램 피드
- Format: Square (1080×1080)
- Background: Dark Blue, Dark Magenta
- Title: 앨범명

### 스토리/릴스
- Format: Mobile (1080×1920)
- Background: Purple Gradient, Dark Cyan
- Title: 아티스트명 or 곡명

### 틱톡
- Format: Mobile (1080×1920)
- Background: 트렌디한 색상
- Title: #해시태그
